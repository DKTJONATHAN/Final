Here's a **general-purpose Markdown template** for your Astro blog posts with all key sections and frontmatter:

```markdown
---
title: "Your Post Title"
pubDate: YYYY-MM-DD
description: "1-2 sentence summary of the post"
author: "Your Name"  # Optional
tags: ["tag1", "tag2"]  # Max 3-5 tags
image:
  src: "/images/your-image.jpg"  # Path from /public
  alt: "Image description"  # Required for accessibility
  width: 1200  # Optional dimensions
  height: 630
draft: false  # Set to true for unpublished posts
---

### **Template Features**:
1. **Frontmatter**  
   - Matches your `content.config.ts` schema  
   - SEO-friendly with image meta  
   - Draft mode for unpublished posts  

2. **Body Structure**  
   - Semantic headers (H2, H3)  
   - Code blocks with syntax highlighting  
   - Tables, footnotes, and blockquotes  
   - Responsive image syntax  

3. **Best Practices**  
   - 80-100 characters per line  
   - Two spaces between sections  
   - Descriptive alt text for images  

Save this as `post-template.md` in your project's docs folder for reference when creating new posts.
